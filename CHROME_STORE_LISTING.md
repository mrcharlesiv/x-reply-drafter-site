## Permissions Justification

**Permissions in manifest.json:**

1. **`activeTab`** - Required to read current X.com tab
2. **`scripting`** - Required to inject the "Draft Reply" button
3. **`host_permissions: ["https://x.com/*", "https://twitter.com/*"]`** - Required to work on X
4. **`storage`** - Required to save user settings and API keys locally

**Why no data collection?**
- All settings stored locally in chrome.storage
- No external servers storing user data
- API key stored encrypted on user's machine
